Riponda
-------------------------------

To import Riponda, you can either use:
```python
from psychopy_cedrus.riponda import Riponda
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.hardware.cedrus import Riponda
```
