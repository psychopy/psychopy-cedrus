RipondaVoicekey
-------------------------------

To import RipondaVoicekey, you can either use:
```python
from psychopy_cedrus.riponda import RipondaVoicekey
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.hardware.cedrus import RipondaVoicekey
```
