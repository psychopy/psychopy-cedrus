RipondaLightSensorGroup
-------------------------------

To import RipondaLightSensorGroup, you can either use:
```python
from psychopy_cedrus.riponda import RipondaLightSensorGroup
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.hardware.cedrus import RipondaLightSensorGroup
```
