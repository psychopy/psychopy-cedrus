RipondaButtonGroup
-------------------------------

To import RipondaButtonGroup, you can either use:
```python
from psychopy_cedrus.riponda import RipondaButtonGroup
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.hardware.cedrus import RipondaButtonGroup
```
